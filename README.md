Projek IOT by Nicholas Stancio Saka

# Instructions (Running locally)
1. run 'npm install'
2. Change base url to 'http://127.0.0.1:1880' in config file
3. open node-red
4. run 'npm run start-dev'
5. live server available on 'http://localhost:8080/'
